# x402-testnet-server

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![npm](https://img.shields.io/npm/v/agentpay-mcp)](https://www.npmjs.com/package/agentpay-mcp)

**X402 testnet server** for developing and testing agent-to-agent payment flows. Implements the [X402 payment protocol](https://x402.org) on testnet for safe experimentation.

## Installation

```bash
git clone https://github.com/up2itnow0822/x402-testnet-server.git
cd x402-testnet-server
npm install
```

## Usage

```bash
npm start
# Server starts at http://localhost:3000
```

## Ecosystem

- [agent-wallet-sdk](https://github.com/up2itnow0822/agent-wallet-sdk) — Non-custodial agent wallets with x402 support (`npm install agentwallet-sdk`)
- [agentpay-mcp](https://github.com/up2itnow0822/agentpay-mcp) — MCP server for agent payments (`npm install agentpay-mcp`)
- [webmcp-sdk](https://github.com/up2itnow0822/webmcp-sdk) — Browser-native WebMCP integration (`npm install webmcp-sdk`)
